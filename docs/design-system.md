# Design System

## Colors

Use the `.modern` theme defined in `globals.css`.

## Typography

| Usage | Font | Fallback |
|-------|------|----------|
| Headings | Space Grotesk | sans-serif |
| Body | Noto Sans | sans-serif |
| Code / Terminal | JetBrains Mono | monospace |

Font weights:
- Headings: 500-700
- Body: 400-500
- Code: 400
